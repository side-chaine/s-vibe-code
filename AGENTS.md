## Алгоритм при получении задачи (если триггер НЕ распознан)
1. Определи номер задачи (смотри какие папки уже есть в `context/tasks/`, +1)
2. Определи домен (какую часть проекта затрагивает)
3. Если нужно — grep/read из справочников по домену
4. Создай папку `context/tasks/[номер] [Название]/`
5. Внутри создай `[номер]-MACRO.md`
6. В MACRO-файл упакуй ПОЛНЫЙ КОД релевантных файлов:

```markdown
# 📦 PACK: [номер] [Название]

## 📄 src/path/to/file.ts (полный файл)
```tsx
// ... весь код ...
```

## 📄 src/path/to/next.ts (полный файл)
```tsx
// ... весь код ...
```
```

7. Скажи: *"Пакет готов в папке [номер] [Название]. Неси Центру."*
